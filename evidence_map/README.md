# Evidence Map

This folder will contain structured evidence tables for the AI/ML WiFi Sensing Hub.

The evidence map connects papers, datasets, code releases, reproducibility status, application categories, security relevance, and collaboration value.

## Planned files

| File | Purpose |
|---|---|
| `papers_index.csv` | Structured paper list |
| `datasets_index.csv` | Structured dataset list |
| `code_releases_index.csv` | Structured code repository list |
| `security_gap_map.csv` | Security, robustness, and privacy evidence |
| `collaboration_candidates.csv` | Research groups, labs, authors, and outreach candidates |

## Purpose

This folder will become the structured database layer of the hub. The GitHub Project board tracks workflow status, while this folder stores curated research evidence in reusable table formats.
