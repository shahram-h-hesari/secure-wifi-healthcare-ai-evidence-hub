# Papers

This folder organizes research papers related to AI/ML WiFi sensing.

The first focus area is healthcare-relevant WiFi sensing, including contactless activity recognition, fall detection, respiration monitoring, vital-sign-related sensing, assisted-living research, and care-aware environments.

## Planned paper categories

| Category | Purpose |
|---|---|
| Core Papers | Foundational or highly relevant papers for the hub |
| Healthcare-Relevant WiFi Sensing | Papers focused on healthcare, care-aware, or contactless monitoring applications |
| Security and Robustness | Papers related to adversarial attacks, spoofing, privacy, robustness, and physical-layer risks |
| Surveys and Reviews | Review papers, benchmark papers, and broad field summaries |
| Future Categories | Smart environments, human activity recognition, robotics, IoT sensing, and other WiFi sensing areas |

## Review fields

Each paper should eventually include:

- Title
- Authors
- Year and venue
- DOI or URL
- Application category
- Dataset used
- Code availability
- Security or robustness relevance
- Reproducibility notes
- Reason for inclusion
