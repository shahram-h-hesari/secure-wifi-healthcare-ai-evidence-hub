# Datasets

This folder tracks datasets related to AI/ML WiFi sensing.

The purpose is to document dataset availability, access status, licensing, modalities, application categories, and suitability for reproducible experiments.

## Dataset tracking goals

| Field | Purpose |
|---|---|
| Dataset Name | Name of the dataset or benchmark |
| Link | Official dataset page, repository, or access request page |
| Access Status | Public, request needed, unavailable, broken link, or unknown |
| Modality | CSI, RSSI, WiFi signal data, multimodal sensing, or other |
| Application Category | Healthcare-relevant sensing, activity recognition, smart environments, etc. |
| Related Papers | Papers that use or introduce the dataset |
| License Notes | Usage restrictions and citation requirements |
| Experiment Suitability | Whether the dataset may support reproducible AI/ML WiFi sensing experiments |

## Important note

Large datasets should not be copied directly into this repository unless licensing, storage, and GitHub size limits are reviewed. This folder is intended first as a structured index of dataset links, access notes, and reproducibility information.
